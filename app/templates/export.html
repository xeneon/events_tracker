ⓘ How to set up Google Sheets access

A service account is a special Google account that lets this app write to your sheet automatically. Follow these steps once:

  1. Go to console.cloud.google.com and sign in with your Google account.
  2. Create a new project (click the project dropdown at the top → New Project), or select an existing one.
  3. In the top search bar, search for Google Sheets API and click Enable. Then do the same for Google Drive API.
  4. In the left menu go to IAM & Admin → Service Accounts. Click Create Service Account, give it any name, and click Done.
  5. Click on the service account you just created, open the Keys tab, then click Add Key → Create new key. Choose JSON and click Create — a file will download to your computer.
  6. Open your Google Sheet and click Share. Add the service account's email address (it's inside the downloaded JSON file, labeled client_email) and give it Editor access.
  7. Upload that downloaded JSON file using the button below.

ⓘ What the default query does
  • Date range: Events from the current month through the end of the current year.
  • Popularity filter: Within each category, keeps the top 15 events by popularity score, plus any event with a score ≥ 50 (on a 0–100 scale).
  • Columns exported: year_month, start_date, category, title, description, popularity_score (0–1 scale), source_url.
  • Custom query: You can write any SQL that returns the columns you need — the column names become the sheet header row automatically.

Column names from the result are used as the sheet header row.
